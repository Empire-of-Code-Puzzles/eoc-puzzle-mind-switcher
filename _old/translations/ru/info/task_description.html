"Futurama" - Season 6 Episode 10 - "The Prisoner of Benda"

Дроны Софии, эти простые роботы ежедневно делают опасную работу Но на случай поломки у Софии есть машина для снятия запасных копий. Эта машина может копировать разум робота, но у нее есть другая недокументированая функция. Данный агрегат может работать, как машина для обмена разумами.

Однажды София забыла выключить копировальную машину и оставила дронов в той же комнате. Когда она вернулась, некоторые из них уже обменялись сознаниями друг с другом. Наша задача вернуть все как было, иначе гарантия будет недействительна.

У нас есть журнал событий о том какие дроны обменялись телами. Но есть одна проблема - если пара дронов обменялась разумами, то обратно этим же телам не поменяться. Однако, у нас есть два дополнительных тела, чтобы упорядочить этот хаос. Никола и София выступили добровольцами, чтобы помочь ("nikola" и "sophia").

Вам дан массив с информацией о прошлых обменах разумами. Каждая запись - это набор двух имен роботов (тел). Вы можете работать с телами, которые указаны в журнале и плюс два дополнительных тела - "nikola" и "sophia". Найдите последовательность обменов разумами, так чтобы вернуть всех в их тела. Результат должен быть в виде массива (list/tuple) наборов (set) с двумя именами роботов (тела) в каждом.

Входные данные: Журнал, как кортеж (tuple) наборов (set). Каждый набор содержит две строки (str, unicode).

Выходные данные: Последовательность действий, как массив (list/tuple) наборов (set). Каждый набор содержит две строки (str, unicode).

Примеры: 

mind_switcher(({"scout", "super"},)) == ({"super", "nikola"}, {"sophia", "scout"},
                                         {"nikola", "scout"}, {"sophia", "super"},
                                         {"nikola", "sophia"})

Зачем это нужно: теорема из данной задачи (точнее эпизода Футурамы) основана на теории груп. Данная теория имеет широкое применение в криптографии, физики и химии.

Предусловия:
0 < len(journal) ≤ 55
journal правилен.
